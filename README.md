# PiHut Let it Glow Advent Calendar 2025

This repo contains the code used for each day of the 12 days of Let it Glow, an Advent Calendar by PiHut.

| Day |  Activity | Tutorial |
|:----------:|:----------|:----------|
| 01 | Download IDE, wire up the breadboard and turn in-build LED on. | [Day 1 guide](https://thepihut.com/blogs/raspberry-pi-tutorials/let-it-glow-advent-calendar-day-1-lets-get-started) |
| 02 | Wire additional LED and make it flash. | [Day 2 guide](https://thepihut.com/blogs/raspberry-pi-tutorials/let-it-glow-maker-advent-calendar-day-2-lighting-leds) |
| 03 | Adding inputs to turn lights on and off | [Day 3 guide](https://thepihut.com/blogs/raspberry-pi-tutorials/let-it-glow-maker-advent-calendar-day-3-incredible-inputs) |
